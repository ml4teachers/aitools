# Microsoft Copilot

## Überblick
[Microsoft Copilot](https://copilot.microsoft.com) ist ein eigenständiges Tool, das Bing Chat ersetzt. Es unterscheidet sich von Microsoft 365 Copilot, welches eine Integration in die Microsoft 365 Suite ermöglicht. Microsoft Copilot bietet Zugriff auf das leistungsstarke [GPT-4 Modell](/tools/chatgptplus) und ermöglicht eine Vielzahl von Funktionen, ohne dass Nutzerinnen und Nutzer ein zusätzliches Login erstellen müssen.

<MDXImage src="/screenshots/copilot-screenshot.png" alt="Microsoft Copilot Benutzeroberfläche" />

## Vorteile
Die Hauptvorteile von Microsoft Copilot gegenüber der kostenlosen ChatGPT-Variante sind:
- Die Eingaben der Nutzerinnen und Nutzer werden nicht für das weitere Training der Modelle verwendet.
- Es ist kein zusätzliches Login erforderlich.
- Zugriff auf das fortschrittlichere GPT-4 Modell.


Um die leistungsfähigere Variante von Microsoft Copilot, die auf GPT-4 basiert, zu nutzen, müssen Nutzerinnen und Nutzer den Unterhaltungsstil "Im höheren Maße Kreativ" auswählen. Diese Einstellung aktiviert die fortgeschrittenen Funktionen des GPT-4 Modells. Wird hingegen "Im höheren Maße Ausgewogen" gewählt, verwendet Copilot das GPT-3.5 Modell, welches der kostenlosen ChatGPT-Version entspricht. Die Einstellung "Im höheren Maße Genau" fokussiert sich auf präzise und detaillierte Antworten, die weniger kreativen Spielraum bieten.

<MDXImage src="/screenshots/copilot-styles.png" alt="Auswahl des Unterhaltungsstils bei Microsoft Copilot" />

## Einschränkungen
Ein Nachteil von Microsoft Copilot ist, dass es keine "Custom Instructions" unterstützt und dass die Chat-History nicht gespeichert wird. Dies kann die Kontinuität bei wiederholter Nutzung beeinträchtigen.

## Dall-E 3 Integration
Ein bemerkenswertes Merkmal von Microsoft Copilot ist der Zugriff auf Dall-E 3. Nutzerinnen und Nutzer können Bilder durch Texteingaben generieren, indem sie Stichworte wie "generiere" oder "zeichne" verwenden. Die so erstellten Bilder können urheberrechtsfrei in Arbeitsblättern, Präsentationen und anderen Materialien verwendet werden.

<MDXImage src="/screenshots/copilot-dalle3.png" alt="Bildgenerierung mit Microsoft Copilot und Dall-E 3" />

## Anwendungsmöglichkeiten im Bildungsbereich
Lehrpersonen können Microsoft Copilot auf vielfältige Weise nutzen, darunter:
- Erstellen von Unterrichtsmaterialien und Prüfungen.
- Unterstützung bei der Erstellung von Präsentationen.
- Generierung von Bildmaterial für Lehrzwecke.
- Schnelle und effiziente Beantwortung fachspezifischer Fragen.

## Datenschutz
Wie bereits erwähnt, werden die Eingaben der Nutzerinnen und Nutzer bei Microsoft Copilot nicht für das Training weiterer Modelle verwendet, was den Datenschutz erhöht. Nutzerinnen und Nutzer können daher sicher sein, dass ihre Daten vertraulich behandelt werden.
