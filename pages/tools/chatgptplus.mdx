# ChatGPT Plus

[ChatGPT Plus](https://openai.com/blog/chatgpt-plus) bietet für eine monatliche Gebühr von CHF 20.- einen erweiterten Zugang zu den neuesten Funktionen von OpenAI's ChatGPT. Dazu gehören nicht nur das fortschrittliche **GPT-4(V)** Modell, sondern auch die Integration von **Dall-E 3** und eine Konversationsfunktion in natürlicher Sprache für die Smartphone-Applikation. Mit zusätzlichen Funktionen wie der **Advanced Data Analysis** und **Plug-Ins** bietet ChatGPT Plus eine breitere Palette an Möglichkeiten und eine verbesserte Genauigkeit.

## Vorteile von GPT-4(V)

Gegenüber GPT-3.5 bietet GPT-4 verbesserte Funktionen:
- **Erweitertes Wissensspektrum**: GPT-4 hat dank der Nutzung von Plug-Ins Zugriff auf aktuellere Informationen und bietet somit fundiertere Antworten auf aktuelle Fragen.
- **Bessere Semantikerfassung**: Das Modell versteht die Bedeutung von Wörtern und Sätzen besser, was zu präziseren Antworten führt.
- **Größeres Kontextfenster**: GPT-4 kann längere Gespräche besser verfolgen und bietet konsistentere Antworten über längere Dialoge hinweg.
- **Bildanalyse-Fähigkeiten**: GPT-4V kann nun auch Bilder analysieren und verstehen, was die Interaktion vielseitiger macht und einen breiteren Anwendungsbereich ermöglicht.

## Dall-E 3 Integration
Die Integration von Dall-E 3 in ChatGPT vereinfacht die Erstellung von Bildmaterial. Nutzende können einfach in natürlicher Sprache beschreiben, was sie auf ihrem Bild sehen möchten. GPT-4 konvertiert diese Beschreibung in einen Prompt für Dall-E 3, das dann vier mögliche Bildvarianten generiert. Diese Funktion ist besonders nützlich für die Erstellung von Folienhintergründen, lizenzfreien Bildern für Arbeitsblätter oder Webseiten.


## Konversationsfunktion für die iOS und Android-Applikation

Diese Funktion ermöglicht eine Kommunikation mit ChatGPT in gesprochener Sprache. Anstatt auf geschriebene Eingaben angewiesen zu sein, können Nutzende eine natürlichsprachliche Interaktion mit dem Modell durchführen und gesprochene Antworten erhalten. Dies unterstützt eine tiefgehende Auseinandersetzung mit vielschichtigen Fragestellungen, sowohl im schulischen Kontext als auch darüber hinaus.

## Plug-Ins

Die [Plug-In-Funktion](https://openai.com/blog/chatgpt-plugins) erweitert die Fähigkeiten von ChatGPT erheblich:
- **Zugriff auf Suchmaschinen**: Das Modell kann in Echtzeit aktuelle Informationen aus dem Web abrufen.
- **Lesen von PDF-Dateien**: ChatGPT kann den Inhalt von PDF-Dateien durchsuchen und daraus relevante Informationen extrahieren.
- **Taschenrechnerfunktionen**: Für mathematische Abfragen kann das Modell präzise Berechnungen durchführen.

## Advanced Data Analysis

Die Funktion **Advanved Data Analysis** (ehemals Code Interpreter) ist eine Erweiterung für ChatGPT Plus, die es dem Modell ermöglicht, in einem persistenten Arbeitsbereich Programme zu schreiben, auszuführen und Informationen hoch- und herunterzuladen. Die folgenden Tipps stammen von einem Artikel von [Ethan Mollick](https://www.oneusefulthing.org/p/what-ai-can-do-with-a-toolbox-getting), Associate Professor an der Wharton School der University of Pennsylvania und Experte für Innovation:

1. **Datenanalyse**: Der Code Interpreter kann Daten analysieren, verarbeiten und präsentieren. Dies automatisiert viele der komplexen Aspekte der quantitativen Analyse.

2. **Direktes Arbeiten mit Python**: Der Code Interpreter ermöglicht es ChatGPT, Python-Code zu schreiben und auszuführen, was eine Vielzahl von Problemen lösen kann, von mathematischen Berechnungen bis hin zur Datenvisualisierung.

3. **Verbesserte Genauigkeit**: Durch das Arbeiten mit Python-Code werden Halluzinationen reduziert. Python generiert Fehlermeldungen, wenn der Code inkorrekt ist.

4. **Mehr Interaktivität**: Nutzer können mit dem Modell interagieren und Änderungen in Echtzeit anfordern. Dies bietet ein hohes Maß an Flexibilität und Anpassungsfähigkeit.

5. **Vereinfachte Datenverarbeitung**: Der Code Interpreter kann Daten hochladen, bereinigen und verarbeiten, wodurch viele der mühsamen Schritte in der Datenanalyse automatisiert werden.

6. **Intelligente Dateninterpretation**: Das Modell kann tiefgreifende Analysen durchführen und dabei interessante Schlussfolgerungen ziehen.

## Bildanalyse-Fähigkeiten mit GPT-4V

<MDXImage src="/screenshots/gpt-4v-screenshot.jpg" alt="ChatGPT versteht teilweise Bildinhalte" />

Die Möglichkeit, Bilder zu analysieren, eröffnet GPT-4V zusätzliche Anwendungsgebiete, insbesondere im pädagogischen Kontext:

1. **Aufgabenunterstützung für Kinder**: Lehrpersonen können Bilder von Arbeitsblättern oder spezifischen Aufgabenstellungen hochladen. GPT-4V kann diese analysieren und konkrete Hilfestellungen oder Lösungsvorschläge bieten. Ein Beispiel dafür ist im Screenshot zu sehen, wo GPT-4V eine Aufgabe zum Thema Binärcode erfolgreich interpretiert hat.

2. **Verstehen komplexer Grafiken**: Grafiken, Diagramme oder andere visuelle Darstellungen können gerade in naturwissenschaftlichen Fächern sehr komplex sein. GPT-4V kann helfen, diese verständlicher zu machen.

3. **Automatische Bildbeschreibung**: Bei der Erstellung von Unterrichtsmaterialien kann GPT-4V beim schnellen Generieren von Bildbeschreibungen oder Alt-Texten behilflich sein, was die Zugänglichkeit der Materialien verbessert.

Es ist jedoch wichtig zu betonen, dass die Bildanalyse, genau wie die Textanalyse von ChatGPT, Fehler generieren kann und nicht zu 100% verlässlich ist. Die Verantwortung und Überprüfung liegt stets bei der Lehrperson. Für die Ideenfindung und erste Analysen ist diese Funktion jedoch ein wertvolles Tool.


## Datenschutz

Eingaben der Nutzenden werden zur Verbesserung der aktuellen Modelle in zukünftigen Trainingsprozessen gespeichert. Persönliche oder vertrauliche Daten sollten nicht geteilt werden. In den Einstellungen unter Data Controls kann das Training deaktiviert werden, um den Datenschutz zu gewährleisten.
